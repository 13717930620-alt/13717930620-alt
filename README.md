## 🔌 Core Products

### [UAA — Universal Agent Adapter](https://github.com/13717930620-alt/UAA) ⭐ Open Source
> Safely connect AI agents to any physical device.

TypeScript middleware that translates AI agent intents into device-specific commands across Modbus TCP, OS-exec, and HTTP API protocols. Features a 3-level safety guardrail (parameter bounds, state preconditions, conflict detection), digital twin state tracking, and emergency stop with 0ms response.

**`npm install` · TypeScript · Apache 2.0**

### Somaagent — AI Agent Framework
> Multi-provider AI agent engine with 4-phase orchestration.

Dual-protocol support for Anthropic API (Claude Opus/Sonnet/Haiku) and OpenAI-compatible providers (Ollama, DeepSeek, Groq). 7 built-in lazy-loaded skills: Web Search, Word, Excel, PowerPoint, PDF, File System, Browser Automation.

### Human Brain Project — v8.0
> 19-region brain-inspired cognitive architecture.

Biologically-modeled autonomous agent with global workspace consciousness, active inference (Free Energy Principle), dopamine reward system, amygdala threat assessment, and 10-dimension self-evolution engine. Engineer and Legal variants available.
